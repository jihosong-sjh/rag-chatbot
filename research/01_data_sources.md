## 1. 금감원 전자공시시스템 (DART / OPENDART)

### 개요

금융감독원이 운영하는 전자공시시스템(DART)은 기업의 공시서류를 전자적으로 제출·공개하는 시스템이다. 보험회사도 금융회사로서 사업보고서, 분기보고서 등을 DART를 통해 공시한다.

### URL / 접근 경로

| 구분 | URL |
|------|-----|
| DART 웹사이트 | https://dart.fss.or.kr/ |
| OPENDART (오픈API) | https://opendart.fss.or.kr/ |
| OPENDART 개발가이드 | https://opendart.fss.or.kr/guide/main.do?apiGrpCd=DS001 |
| 인증키 신청 | https://opendart.fss.or.kr/uss/umt/EgovMberInsertView.do |

### 제공 데이터 형태

- **공시보고서 원문**: XML 형식으로 다운로드 가능
- **재무제표 등 핵심 데이터**: JSON/XML API 제공
- **첨부파일**: 공시보고서에 첨부된 파일(PDF, Excel 등) 다운로드 API 존재

### API 현황

OPENDART API는 무료로 제공되며, 인증키 발급 후 사용 가능하다.

- **공시정보 API**: 공시목록 조회, 기업개황 조회
- **사업보고서 주요정보 API**: 재무제표, 배당 등
- **첨부파일 다운로드 API**: 공시보고서 첨부 파일 일괄 다운로드 (ZIP)
- **일일 요청 제한**: 기본 10,000건/일 (추가 신청 가능)
- **Python 라이브러리**: [OpenDartReader](https://github.com/FinanceData/OpenDartReader), [dart-fss](https://dart-fss.readthedocs.io/) 등 오픈소스 존재

### 보험약관 확보 가능성

- **직접적 약관 확보는 어려움**: DART는 주로 기업 재무정보 공시 시스템이므로, 보험약관 자체를 별도로 공시하지는 않음
- **간접 확보 가능성**: 보험회사의 사업보고서 내 "주요 상품 현황" 등에서 약관 관련 요약 정보를 확인할 수 있으나, 전문(全文) 약관 PDF를 직접 제공하지는 않음
- **활용 가치**: 보험회사 기본정보(회사명, 사업자번호 등) 및 재무정보 확보에 유용

### 크롤링/다운로드 가능 여부

- OPENDART API를 통한 합법적 데이터 접근 가능 (무료)
- robots.txt 준수 필요

### 라이선스/저작권

- 공공데이터로 분류되어 자유롭게 활용 가능
- 다만 API 이용약관 준수 필요 (상업적 이용 가능)

## 2. 보험개발원 (KIDI)

### 개요

보험개발원(Korea Insurance Development Institute, KIDI)은 보험요율 산출, 보험통계 관리, 소비자 정보 제공 등을 담당하는 공공기관이다.

### URL / 접근 경로

| 구분 | URL |
|------|-----|
| 보험개발원 홈페이지 | https://www.kidi.or.kr/ |
| 보험정보 빅데이터 플랫폼 (BIGIN) | https://bigin.kidi.or.kr:9443/ |
| 보험통계조회서비스 (INCOS) | https://incos.kidi.or.kr:5443/ |
| 공공데이터포털 (보험개발원) | https://www.data.go.kr/ (보험개발원 검색) |

### 제공 데이터 형태

- **보험통계**: 월별/분기별 보험 통계 데이터 (웹 조회, Excel 다운로드)
- **차량기준가액**: 자동차보험 관련 차량 가격 데이터
- **참고순보험요율**: 보험요율 관련 데이터
- **BIGIN 플랫폼**: 보험 빅데이터 분석 서비스

### 보험약관 확보 가능성

- **약관 직접 제공은 제한적**: 보험개발원은 주로 통계/요율 데이터를 제공하며, 개별 보험약관 PDF를 직접 제공하는 서비스는 확인되지 않음
- **참고순보험요율 관련 문서**: 표준화된 보험요율 산출 기준 문서는 접근 가능할 수 있으나, 이는 약관과는 다른 성격의 자료
- **공공데이터포털 현황**: 보험개발원이 data.go.kr에 제공하는 API는 현재 1건(실손24 참여 요양기관 안내 서비스)으로 매우 제한적

### 대략적 데이터 양

- 보험통계 데이터: 생명보험/손해보험 월별 통계 (수십 년치 축적)
- 약관 관련 데이터: 직접 제공하지 않음

### 크롤링/다운로드 가능 여부

- BIGIN 플랫폼에서 일부 데이터 다운로드 가능 (회원가입 필요)
- 보험통계 조회는 웹 기반으로 제공, 대량 다운로드는 제한적

### 라이선스/저작권

- 공공기관 데이터이나, 이용약관 확인 필요
- BIGIN 플랫폼은 별도 이용약관 존재

## 3. 생명보험협회

### 개요

생명보험협회는 국내 생명보험회사들의 협회로, 상품공시실을 통해 각 보험사의 상품 정보, 약관, 상품요약서 등을 공개한다.

### URL / 접근 경로

| 구분 | URL |
|------|-----|
| 생명보험협회 홈페이지 | https://www.klia.or.kr/ |
| **공시실 (핵심)** | https://pub.insure.or.kr/ |
| 내보험찾아줌 | https://cont.insure.or.kr/ |

### 제공 데이터 형태

공시실(pub.insure.or.kr)에서 다음 정보를 제공한다:

#### 상품비교공시 (보장성)
- 종신보험, 정기보험, 질병보험, 암보험, CI보험
- 상해보험, 어린이보험, 치아보험, 간병/치매보험

#### 상품비교공시 (저축성)
- 연금보험, 저축보험, 교육보험

#### 특화 상품
- 변액보험, 퇴직연금, 연금저축, 실손의료보험

#### 제공 자료 유형
- **약관**: PDF 파일 형태로 다운로드 가능
- **상품요약서**: PDF 파일
- **사업방법서**: PDF 파일
- **상품비교 데이터**: 웹 테이블 + Excel 다운로드 지원

### 보험약관 확보 가능성

- **가장 유력한 소스 중 하나**: 생명보험 전 상품에 대한 약관 PDF를 공시 의무에 따라 제공
- **회사별/상품별 약관 조회 가능**: 보험사와 상품을 선택하여 해당 약관 PDF를 열람/다운로드 가능
- **공시 의무 자료**: 금감원 및 생명보험협회 공시기준에 따라 표준화된 양식으로 제공

### 대략적 데이터 양

- 국내 생명보험사 약 22개사 (2025년 기준)
- 각 회사별 수십~수백 개 상품
- 추정 약관 PDF 수: 수천~만 건 이상 (현재 판매 중 + 과거 판매 상품)

### 크롤링/다운로드 가능 여부

- **PDF 다운로드**: 개별 상품별로 약관 PDF 다운로드 가능 (웹 UI를 통해)
- **일괄 다운로드 API 없음**: 공개 API는 제공되지 않으며, 웹 UI를 통한 개별 조회만 가능
- **크롤링 가능성**: 웹 스크래핑을 통해 자동화 가능하나, 동적 페이지(JavaScript 렌더링) 처리 필요할 수 있음
- **주의사항**: 과도한 요청 시 IP 차단 가능성 있음, 적절한 요청 간격 설정 필요

### 라이선스/저작권

- 공시 의무에 따라 공개된 자료이므로 열람은 자유
- 약관 자체는 보험사의 저작물이나, 공시 목적으로 공개되어 있어 정보 활용 목적의 사용은 가능할 것으로 판단
- 상업적 이용 시 법적 검토 권장 (아래 저작권 섹션 참조)

## 4. 손해보험협회

### 개요

손해보험협회는 국내 손해보험회사들의 협회로, 손해보험 상품의 공시, 비교, 약관 열람 서비스를 제공한다.

### URL / 접근 경로

| 구분 | URL |
|------|-----|
| 손해보험협회 홈페이지 | https://www.knia.or.kr/ |
| **공시실 (핵심)** | https://kpub.knia.or.kr/ |
| 소비자포털 | https://consumer.knia.or.kr/ |
| 자동차보험 정보 | https://carinfo.knia.or.kr/ |

### 제공 데이터 형태

공시실(kpub.knia.or.kr)에서 다음 정보를 제공한다:

#### 상품비교공시 (보장성)
- 화재보험, 종합보험, 상해보험(운전자/기타)
- 질병보험, 암보험, 어린이보험, 치아보험
- 간병/치매보험, 비용보험, 반려동물보험

#### 상품비교공시 (저축성)
- 장기화재/종합, 장기상해, 연금저축, 퇴직연금

#### 특화 상품
- 실손의료보험, 자동차보험

#### 제공 자료 유형
- **약관**: PDF 파일 형태로 다운로드 가능
- **상품요약서(핵심설명서)**: PDF 파일
- **상품비교 데이터**: 웹 테이블 형태

### 보험약관 확보 가능성

- **생명보험협회와 함께 가장 유력한 소스**: 손해보험 전 상품에 대한 약관 PDF를 공시 의무에 따라 제공
- **회사별/상품별 약관 조회 가능**: 보험사와 상품을 선택하여 해당 약관 PDF를 열람/다운로드 가능

### 대략적 데이터 양

- 국내 손해보험사 약 18~20개사
- 각 회사별 수십~수백 개 상품 (손해보험은 상품 종류가 특히 다양)
- 자동차보험 약관 별도 존재
- 추정 약관 PDF 수: 수천~만 건 이상

### 크롤링/다운로드 가능 여부

- **PDF 다운로드**: 개별 상품별 약관 PDF 다운로드 가능 (웹 UI)
- **일괄 다운로드 API 없음**: 공개 API 미제공
- **크롤링 가능성**: 웹 스크래핑을 통한 자동화 가능, 동적 페이지 처리 필요할 수 있음
- **주의사항**: 생명보험협회와 동일하게 과도한 요청 시 차단 가능성

### 라이선스/저작권

- 생명보험협회와 동일한 조건 적용
- 공시 의무에 따라 공개된 자료

## 7. 권장 데이터 확보 전략

### Phase 1: 표준약관 확보 (1주)

1. **금감원 표준약관 수집**: 금융감독원 홈페이지에서 보험종류별 표준약관 PDF/HWP 파일 수동 다운로드
2. **국가법령정보센터**: 보험업감독규정 및 시행세칙 별표15(표준약관) 확보
3. 데이터 양: 약 10~15건 (소량이므로 수동 수집 가능)

### Phase 2: 협회 공시실 크롤링 (2~3주)

1. **생명보험협회 공시실 크롤러 개발**
   - 대상: https://pub.insure.or.kr/
   - 회사별 > 상품별 약관 PDF 다운로드 자동화
   - Selenium/Playwright 등 동적 페이지 처리 도구 필요 가능성
   - 적절한 요청 간격 설정 (1~3초/건)

2. **손해보험협회 공시실 크롤러 개발**
   - 대상: https://kpub.knia.or.kr/
   - 동일한 접근 방식

### Phase 3: 보충 데이터 확보 (선택)

1. **주요 보험사 홈페이지**: 협회 공시실에서 누락된 상품이 있을 경우 보충
2. **DART API**: 보험회사 기본정보 및 재무정보 보조 데이터 수집

### 법적/윤리적 고려사항

#### 크롤링 합법성

한국법상 크롤링의 위법성은 다음 기준으로 판단된다:

1. **공개된 정보 여부**: 보험약관은 공시 의무에 따라 누구에게나 공개된 정보이므로 비교적 안전
2. **이용약관 위반 여부**: 각 사이트의 이용약관에서 자동화된 접근을 금지하는지 확인 필요
3. **서버 부하**: 과도한 요청으로 서비스에 지장을 주지 않도록 적절한 간격 유지
4. **데이터베이스권 침해**: 전체 데이터베이스를 체계적으로 복제하는 것은 위험, 필요한 범위 내에서 수집 권장
5. **개인정보**: 보험약관에는 개인정보가 포함되지 않으므로 개인정보보호법 이슈는 낮음

#### 저작권 관련

- **보험약관 자체**: 보험사의 저작물이나, 법률문서적 성격이 강하고 공시 의무에 따라 공개되어 있어 정보 제공 목적의 활용은 비교적 안전
- **표준약관**: 금융위원회 고시로 제정된 것이므로 공공저작물에 해당할 가능성 높음
- **상업적 이용**: RAG 챗봇의 상업적 서비스 시 법적 자문을 받는 것을 권장
- **참고 판례**: 부정경쟁방지법상 "타인의 상당한 투자나 노력으로 만들어진 성과"를 무단 사용하는 것은 부정경쟁행위에 해당할 수 있음 (부경법 제2조 제1호 파목)

#### 권장 조치

1. 각 협회/사이트의 robots.txt 확인 및 준수
2. 크롤링 시 적절한 User-Agent 설정 및 요청 간격 유지
3. 수집 데이터의 사용 목적을 명확히 하고, 필요 시 데이터 제공 기관에 사전 협의
4. 상업적 서비스 출시 전 법률 검토 수행
